This folder contains data that are referred in supplementary material due to space constraints in the main paper.

- `aggregated_results_undirected.csv` contains experiment results of unguided-Cleverest (implemented in `run_undirected.sh`).
- `xml50.env` and `SUMMARY_xml50.txt` contain list of 50 recent non-bug commits automatically selected from Libxml2 v2.13.3 and result summary.
- `stat_46commits.csv` shows commit length statistics about of 46 commits used in the main experiment.
